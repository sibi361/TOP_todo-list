# TOP_todo_list

Todo list web app made as part of [The Odin Project](https://www.theodinproject.com/).

DEMO: https://sibi361.github.io/TOP_todo_list/

## Features

-   Tasks
    -   [ ] Add
    -   [ ] Update
    -   [ ] Delete
    -   [ ] Due date
    -   [ ] Priority
    -   [ ] Tag
-   [ ] Persistent storage in browser's `localStorage`
-   [ ] Import and Export tasks to JSON
